# Distribuzione (Excel-only)

La soluzione Ã¨ progettata per funzionare **solo con Excel + Power Query**.

## Requisiti
- Microsoft Excel con Power Query (Excel 2016+ / Microsoft 365).
- Nessun Python.
- Nessun eseguibile `.exe`.

## Asset di template
Usare i file in `TEMPLATE/`:
- `piano_query.m` (query M)
- `POWER_QUERY_TEMPLATE.md` (istruzioni operative)

## Flusso operativo
1. Mettere i file SAP in `INPUT/`.
2. Aprire il file Excel template.
3. Eseguire **Aggiorna tutto**.
4. Verificare tabella `piano` e (se necessario) rigenerare i fogli cliente.
